    Promote Tests to Manual Tests — Implementation Plan#

    For Claude: REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

    Goal: Let setters freeze the actual input of a generated test or a stress finding into a static, glob-backed manual test group, via a new rbx testcases promote command and a new manual route in stress's post-match prompt.

    Architecture: A shared core module (rbx/box/promotion.py) provides the primitive "write this input into this manual group" plus helpers to resolve a manual group's folder, pick the next filename, and create a new manual group. Two call sites use it: the new promote command in rbx/box/testcases/main.py (interactive + non-interactive), and the existing post-match prompt in rbx/box/cli.py's stress command. All promotions write input only — .out is regenerated by the normal build pipeline for manual tests.

    Tech Stack: Python 3, Typer (CLI), Pydantic v2 (schema), ruyaml (comment-preserving YAML edits), questionary (interactive prompts), pytest.

    Design doc: docs/plans/2026-06-03-promote-tests-to-manual-design.md

    Background facts the implementer needs#

    • A manual group is a TestcaseGroup/TestcaseSubgroup (rbx/box/schema.py:409+) whose testcaseGlob is set (e.g. tests/manual/corner/*.in). The check_oneof validator forbids combining testcaseGlob with generators/generatorScript/testcases, so a manual group is glob-only.
    • package.get_test_groups_by_name() (rbx/box/package.py:509) returns dict[str, TestcaseGroup].
    • TestcaseEntry (rbx/box/testcase_schema.py) is {group, index}; .parse("grp/3") parses the [group]/[index] selector; .get_prefix_path() → build/tests/<group>/<index:03d> (built input is <prefix>.in).
    • TestcasePattern (rbx/box/testcase_utils.py:17) + extract_generation_testcases_from_patterns(...) (rbx/box/testcase_extractors.py:451) resolve one or more selector patterns into GenerationTestcaseEntry objects.
    • In rbx/box/testcases/main.py, _generate_input_for_editing(entry, output=False, progress=...) returns the path to a test's .in, generating it on demand. Reuse this to get input bytes for an existing test.
    • The existing stress "make it stick" prompt is rbx/box/cli.py:996-1071. It lists .txt-script groups + (create new script) + (skip), then appends finding.generator calls to the chosen script. Stress findings' input files live at package.get_problem_runs_dir() / '.stress' / 'findings' / f'{i}.in', where i is the index into report.findings (rbx/box/stresses.py:626).
    • ruyaml edit pattern: ru, problem_yml = package.get_ruyaml(), mutate problem_yml, utils.save_ruyaml(package.find_problem_yaml(), ru, problem_yml), then package_utils.clear_package_cache(). See rbx/box/cli.py:1032-1044 for the (create new script) precedent.
    • Code style: single quotes, absolute imports only, run uv run ruff format . + uv run ruff check --fix . before each commit. Commit with the /commit convention (conventional commits + Co-Authored-By: Claude Opus 4.8 (1M context) <noreply@anthropic.com>).

    import pathlib
from typing import Dict, Optional

from rbx.box import package
from rbx.box import package as package_utils  # alias kept explicit for clarity
from rbx.box.schema import TestcaseGroup


def manual_group_dir(group: TestcaseGroup) -> pathlib.Path:
    """Folder backing a glob-based manual group (the dir portion of its glob)."""
    assert group.testcaseGlob is not None
    return pathlib.Path(group.testcaseGlob).parent


def next_testcase_name(folder: pathlib.Path) -> str:
    """Next free zero-padded counter name (without extension) in *folder*."""
    existing = {p.stem for p in folder.glob('*.in')} if folder.exists() else set()
    i = 0
    while f'{i:03d}' in existing:
        i += 1
    return f'{i:03d}'


def promote_input_to_group(
    input_path: pathlib.Path,
    group: TestcaseGroup,
    *,
    name: Optional[str] = None,
    base_dir: pathlib.Path = pathlib.Path(),
) -> pathlib.Path:
    """Write *input_path* as a static .in into *group*'s folder. Input only."""
    folder = base_dir / manual_group_dir(group)
    folder.mkdir(parents=True, exist_ok=True)
    stem = name if name is not None else next_testcase_name(folder)
    dest = folder / f'{stem}.in'
    dest.write_bytes(pathlib.Path(input_path).read_bytes())
    return dest


def get_manual_groups_by_name() -> Dict[str, TestcaseGroup]:
    """Existing testcase groups that are glob-backed manual groups (*.in)."""
    return {
        name: group
        for name, group in package.get_test_groups_by_name().items()
        if group.testcaseGlob is not None and group.testcaseGlob.endswith('.in')
    }


def create_manual_group(name: str, glob: str) -> TestcaseGroup:
    """Create folder for *glob*, append a glob-backed group to problem.rbx.yml."""
    folder = pathlib.Path(glob).parent
    folder.mkdir(parents=True, exist_ok=True)

    ru, problem_yml = package.get_ruyaml()
    if 'testcases' not in problem_yml:
        problem_yml['testcases'] = []
    problem_yml['testcases'].append({'name': name, 'testcaseGlob': glob})
    dest = package.find_problem_yaml()
    assert dest is not None

    from rbx import utils  # local import to avoid cycle

    utils.save_ruyaml(dest, ru, problem_yml)
    package_utils.clear_package_cache()
    return TestcaseGroup(name=name, testcaseGlob=glob)

    @app.command('promote', help='Promote tests into a manual (glob-backed) test group.')
@package.within_problem
@syncer.sync
async def promote(
    selectors: Annotated[
        Optional[List[str]],
        typer.Argument(help='Tests to promote, each as [group]/[index]. '
                       'If omitted, choose interactively.'),
    ] = None,
    group: Annotated[
        Optional[str],
        typer.Option('--group', '-G', help='Destination manual group name.'),
    ] = None,
    name: Annotated[
        Optional[str],
        typer.Option('--name', '-n', help='Filename stem for a single promoted test.'),
    ] = None,
):
    from rbx.box import promotion

    if selectors:
        # Non-interactive: --group must name an existing manual group.
        manual_groups = promotion.get_manual_groups_by_name()
        if group is None:
            # Fall back to interactive group picker (Task 3).
            group = await _pick_manual_group(manual_groups)
            if group is None:
                return
        if group not in manual_groups:
            console.print(
                f'[error]Manual group [item]{group}[/item] does not exist. '
                f'Create it first (run [item]rbx testcases promote[/item] interactively).[/error]'
            )
            raise typer.Exit(1)
        target = manual_groups[group]

        patterns = [TestcasePattern.parse(s) for s in selectors]
        entries = await extract_generation_testcases_from_patterns(patterns)
        if not entries:
            console.print('[error]No tests matched the given selectors.[/error]')
            raise typer.Exit(1)

        with utils.StatusProgress('Promoting tests...') as s:
            for entry in entries:
                input_path = await _generate_input_for_editing(entry, output=False, progress=s)
                written = promotion.promote_input_to_group(
                    input_path, target, name=name if len(entries) == 1 else None
                )
                console.print(f'Promoted [item]{entry.group_entry}[/item] -> {written}')
        return

    # Interactive path implemented in Task 3.
    await _promote_interactive(group=group)

    async def _pick_manual_group(manual_groups) -> Optional[str]:
    import questionary
    from rbx.box import promotion

    CREATE, SKIP = '(create new manual group)', '(skip)'
    choice = await questionary.select(
        'Choose the manual group to promote into:',
        choices=list(manual_groups) + [CREATE, SKIP],
    ).ask_async()
    if choice in (None, SKIP):
        return None
    if choice == CREATE:
        new_name = await questionary.text('Name for the new manual group:').ask_async()
        glob = await questionary.text(
            'Glob for the new manual group (e.g. tests/manual/corner/*.in):'
        ).ask_async()
        promotion.create_manual_group(new_name, glob)  # mkdir -p + yml entry
        package.clear_package_cache()  # match cli.py import
        return new_name
    return choice


async def _promote_interactive(group: Optional[str]) -> None:
    import questionary
    from rbx.box import promotion

    # Build the list of available tests to promote (reuse the extractors).
    all_entries = await extract_generation_testcases_from_patterns([TestcasePattern.parse('*')])
    selected = await questionary.checkbox(
        'Select tests to promote to manual tests:',
        choices=[str(e.group_entry) for e in all_entries],
    ).ask_async()
    if not selected:
        return

    manual_groups = promotion.get_manual_groups_by_name()
    group = group or await _pick_manual_group(manual_groups)
    if group is None:
        return
    manual_groups = promotion.get_manual_groups_by_name()  # refresh after create
    target = manual_groups[group]

    chosen = [e for e in all_entries if str(e.group_entry) in set(selected)]
    with utils.StatusProgress('Promoting tests...') as s:
        for entry in chosen:
            default = promotion.next_testcase_name(
                promotion.manual_group_dir(target)
            )
            name = await questionary.text(
                f'Filename stem for {entry.group_entry}:', default=default
            ).ask_async()
            input_path = await _generate_input_for_editing(entry, output=False, progress=s)
            written = promotion.promote_input_to_group(input_path, target, name=name or None)
            console.print(f'Promoted [item]{entry.group_entry}[/item] -> {written}')
